# 🗽 New York City Blog

Blog sobre a cidade de Nova York criado como projeto prático do curso
**Front-End Engineer** da [Codecademy](https://www.codecademy.com).
Você pode acessar aqui: https://sololucashan.github.io/nyc-blog


## 📋 Sobre o projeto

Projeto desenvolvido para praticar **Semantic HTML** — uso correto
das tags `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`,
`<aside>`, `<figure>` e `<footer>`.

O stylesheet foi criado do zero com foco em design moderno e responsivo,
utilizando CSS custom properties (variáveis), Flexbox, Grid e dark mode.

## 🛠️ Tecnologias usadas

- HTML5 (Semantic HTML)
- CSS3 (Custom Properties, Flexbox, Grid, Media Queries)
- Google Fonts (Playfair Display + Work Sans)

## ✨ Funcionalidades

- Layout responsivo (mobile, tablet e desktop)
- Dark mode automático (respeita as configurações do sistema)
- Navegação sticky com efeito glassmorphism
- Animações suaves de hover nos cards
